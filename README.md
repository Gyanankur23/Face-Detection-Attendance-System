# Face Detection Attendance System

A mobile-friendly, real-time attendance logging app built with Streamlit and OpenCV. Detects faces from webcam or uploaded images and prompts users to enter their name and roll number. Attendance is logged with timestamps and can be saved to a persistent history tracker.
